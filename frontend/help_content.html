Описание приложения

Приложение Braillehoven позволяет конвертировать файлы формата .musicxml (.mxl) в машиночитаемый формат для слепых .brf по щелчку двух кнопок

Клавиатурные сочетания:

  • Ctrl + O: открыть файл для конвертации
  • Esc: отменить выбор файла
  • Ctrl + K: конвертировать файл
  • Ctrl + V: смена режима (слабозрячий/зрячий)
